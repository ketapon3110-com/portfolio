写真家のポートレイトサイト / 土本友護

Designe / Cording / Wire / Direction

http://yugotsuchimoto.com/
概要
フリーランスで写真家をされている土本友護様のwebサイトを作成させて頂きました。
使用ツール
Figma ・ photoshop ・ VisualStudioCode
制作期間
2ヶ月(2024年2月〜3月中旬)
ワイヤー20日 / デザイン 25日 / コーディング 1ヶ月
対応範囲
企画立案 / カンプ制作 / 文章構成 / レイアウト / デザイン / コーディング /
現状
現在、ポートフォリオサイトあり。知り合いや展示で出会った方、インスタからの顧客がメインとなっている。撮影依頼を企業や個人事業主に幅を広げたい。 主に、ファッション、音楽業界での依頼を希望されている。
課題
  • どんな写真を撮るのか、撮影実績が見やすく、どうやって撮影依頼を すればいいか一目でわかる。
  • 写真の世界観にあった動きやデザイン（ 写真やロゴの表示 ・ハンバーガーボタンの動き ・カーソルの変化 ）を行う。
  • SNSで、個展、展示の開催や、自身のブランドの『All Good』を積極的に発信されているので、SNSへの誘導を行う。

感想
クライアント様の写真の特長である”自然光を用いたナチュラルで中性的な写真”の雰囲気にあった デザインを心がけました。 相違がないよう何度もヒアリングを重ねて、柔軟に対応できるところと、 デザインの面で変えない方が良いところを伝えたり、クライアント様とのコミュニケーションの面でも大変勉強になりました。
