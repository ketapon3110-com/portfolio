技術的SEO・パフォーマンス改善（Core Web Vitals）

Front-end / Performance / Direction

URL：https://www.alo-organic.com/
自社ECサイトにおけるCore Web Vitals（CLS/LCP/INP）と表示速度を改善し、離脱率と検索流入の両方を底上げしたプロジェクト。
使用ツール
Google Analytics 4 ・ PageSpeed Insights ・ Lighthouse ・ Google Search Console ・ Visual Studio Code ・ Excel
言語
HTML / CSS / JavaScript / Liquid
制作期間
調査〜改善設計〜実装〜検証まで継続的に対応（リニューアル期間を起点に段階的に実施）。
対応範囲
現状調査 / 課題整理 / 改善方針策定 / フロント実装 / 計測設計 / 効果検証
概要
自社ECサイトにおいて、表示速度指標（Core Web Vitals）が基準未達となっており、モバイル環境での離脱率が高い状態だった。
SEO担当からも改善の要望はあったが、具体的な技術改善までは手が回っていなかったため、課題特定から改善設計、実装、効果検証までをフロントエンド側で一貫して担当し、表示速度と体験品質の改善を主導した。
背景と課題
PageSpeed Insights のスコアは 5〜9 点と低く、特にモバイル回線での表示に時間がかかっていた。
主な技術的な課題は次のとおり。

・LCP画像の読み込みが遅く、ファーストビューの表示が重い
・画像サイズ未指定によるレイアウトシフト（CLS）が多い
・描画ブロックとなるCSSやJavaScriptが多く、初回表示が遅い
・スマートフォンでの回線状況によって体感速度が大きく低下する構造になっていた
コンテンツは改善されていても、「表示前に離脱するユーザー」が多い状態であり、CV以前の体験にボトルネックがあることが明らかだった。
ゴール
Core Web Vitals の基準達成と、それに伴うユーザー体験と検索流入の改善を目指した。
単にスコアを上げるのではなく、モバイルでの実際の読み込み体感を改善し、直帰率とオーガニック流入の両面で効果が出る状態をゴールとして設定した。
アプローチ

1. 課題の可視化と優先順位付け

PageSpeed Insights や Lighthouse で技術的なボトルネックを洗い出しつつ、GA4で回線別の直帰率・離脱率を確認し、数値に影響が大きいポイントから着手した。
そのうえで、「スコアを上げるための改善」ではなく、「ユーザー体験に与える影響の大きさ」で優先順位を決めた。

2. 改善設計とフロント実装

既存テンプレートとの整合性を保ちながら、次のような改善を段階的に実装した。
・LCP画像の圧縮・WebP化・優先読み込み設定
・画像に width / height を指定し、CLSを抑制
・遅延読み込み設定の見直し（重要要素は先読み、下層要素は遅延）
・不要スクリプトの削減と読み込み位置の整理
・CSS読み込み順の調整と、ファーストビュー優先のスタイル適用
テンプレート制約がある中でも、細かい単位でボトルネックを潰していく方針で進めた。

3. 効果検証とチューニング

改善後は、PageSpeed Insights / Lighthouse でのスコアと、GA4での行動指標を定期的に確認。
表示速度が改善したページから順に、直帰率や商品詳細への遷移率が改善していることを確認し、他ページにも同様の改善パターンを展開していった。
実装で意識したこと
表示速度だけを優先してデザインを崩さないよう、ビジュアル担当と相談しながら、画像の扱いや余白の取り方は維持しつつ、読み込み順や形式を見直す形で調整した。
また、表示速度の改善に加えて、検索エンジンにページの内容を正しく伝えるための構造化データの実装も行った。ページ種別ごとに必要な情報（商品情報・パンくずなど）を整理し、検索エンジンが理解しやすい形でマークアップを設計。テンプレート制約を踏まえつつ、今後の改修やページ追加でも再利用できるよう実装パターンを統一した。
さらに、将来的な改修時にも影響範囲が分かりやすいよう、テンプレート内での役割やコメント整理も行い、他メンバーが保守しやすい実装を意識した。
My Role
SEO担当が全体方針を持つ中で、技術的な課題の洗い出しから改善方針の設計、フロント実装、効果検証までを一貫して担当した。
「SEO施策」としてではなく、「ユーザー体験と売上に直結する基盤改善」として位置づけ、施策優先度の整理や影響範囲の説明も行いながら、段階的なリリースをリードした。
数値改善

Core Web Vitals・速度指標

PageSpeed Insights スコア：5〜9 → 29〜35
CLS / LCP / INP はすべて赤判定から改善。

行動・流入指標

モバイル直帰率：−35pt
オーガニック流入：前年比 212％
主要キーワード検索順位：1位を獲得
改善はCVだけでなく、サイト全体の体験価値向上にもつながった。
